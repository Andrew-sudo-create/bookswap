# BookSwap

BookSwap is an Android application for students and university communities to effortlessly exchange textbooks. Discover great deals on used books, list your own for sale, and connect with others—all within a modern, user-friendly app.

## Features

- **Browse Listings:** See a feed of available textbooks complete with cover info, edition, price, seller name, and contact options.
- **Advanced Search:** Find books by title, author, category, ISBN, or keyword.
- **Contact Sellers:** Easily send inquiries directly through the app and get in touch by email.
- **Book Details:** View detailed information on each book, including descriptions, price, condition, edition, and seller info.
- **Categories and Conditions:** Filter and highlight books by academic category (e.g., Computer Science, Biology) and physical condition (Like New, Good, Fair, etc.).
- **Modern UI:** Built with Material Design 3 for an accessible and responsive experience.
- **Profile Management (Coming Soon):** Maintain your book listings and manage your profile (feature stub present).

## Project Structure

- `app/src/main/java/com/example/bookswap/activities`: Main Android activities (entry points like `MainActivity`, `BookDetailsActivity`)
- `app/src/main/java/com/example/bookswap/fragments`: UI fragments for browsing and profiles
- `app/src/main/java/com/example/bookswap/adapters`: RecyclerView adapters
- `app/src/main/java/com/example/bookswap/models`: Data models (e.g., `Book.kt`)
- `app/src/main/res`: Resources including layout, string, and theme files
